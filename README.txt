Projet ISI3

La javadoc se trouve dans le dossier Javadoc, les sources dans le dossier src, les ressources dans le dossier ressources et les librairies externes dans le dossier lib.
Pour lancer le programme, double cliquez sur le fichier "Robocup Rescue.jar"
Les boutons à gauche de la fenêtre permettent de choisir quel type d'objet ajouter à la carte
Le menu en haut de la fenêtre permet d'ajouter une image de fond, de charger/sauvegarder un fichier XML et de créer un nouveau graphe.
